A Set-UID program is like any other Unix program, except that it has a special marking, which is a single bit called a Set-UID bit. This tells the operating system that when the process is executed it should have the effective user ID of the owner user ID. This is privilege granted also applies to child processes of this program.

#!/bin/bash

chown root $1
chmod 4755 $1
